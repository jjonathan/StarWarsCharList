- [Node](https://nodejs.org)

## Run the project locally

- 1 - Clone 
```sh
$ git clone https://github.com/jjonathan/StarWarsCharList.git
$ cd StarWarsCharList
```

- 2 - Install composer dependencies:

```sh
$ composer install
```

- 3 - Install bower dependencies:

```sh
$ cd public/assets
$ bower install
```
